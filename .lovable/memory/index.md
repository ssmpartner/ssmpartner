# Project Memory

## Core
- Brand: Primary #243e3a (Dark Green), Olive #B3B69C. Background #f5f7f5.
- Typography: 'Neue Alte Grotesk' exclusively, max font-semibold.
- UI: Floating Pill aesthetic, rounded-2xl, shadows, framer-motion vertical drift.
- Tech: Supabase (Auth/DB/Storage), custom i18n (DE default).
- Business: Tied broker for Visana (NOT a temp agency).
- UX: Auto scroll-to-top on route change.
- Security: Agency member contact data is intentionally public.
